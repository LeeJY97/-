## 커밋 규칙

#### 형태

`[{분류}] - {출처} {난이도} {제목}`  
(출처와 난이도는 플랫폼에서 문제를 풀었을 경우에만 작성)

- ex) [스택／큐] - 프로그래머스 lv2 올바른 괄호 (플랫폼 O)
- ex) [그리디] - 거스름돈 (플랫폼 X)
- ex) 풀이 외의 경우 `.` 이나 자유롭게 작성

#### 분류

- `그리디`
- `정렬`
- `스택/큐`
- `해시`
- `소수판별`
- `완전탐색`
- `재귀`
- `DFS/BFS`
- `DP`
- `투포인터`
- `다익스트라`
- `GCD/LCM`

#### 파일 및 폴더 생성

- 분류/문제명
  - 문제명.js (or 문제명.py)
  - README.md (문제와 해설 등을 작성)
